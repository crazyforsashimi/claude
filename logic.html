per-ticker 信号系统 · RSI + 布林校准 + 事件闸门

不用统一阈值,而是每只标的用自己近5年数据校准最适合它的 RSI 阈值 与 布林下轨周期——因为固定 RSI<20/25、固定100布林对不同标的效果天差地别(强趋势股 RSI 碰不到 20、100布林胜率塌)。判定用原始胜率而非重罚小样本的 Wilson 下界,理念是 N 小 = 极端罕见 = 大机会。事件闸门再给买入信号叠一层过滤,避开财报暴雷坑。

🎯 事件闸门 · 财报暴雷坑

所有买入信号之上的过滤层 · 防"接财报雷的飞刀"(v26 新增)

超卖抄底最经典的死法,是接一根财报业绩暴雷砸出来的阴线:好公司某季 miss、股价崩、RSI 瞬间超卖触发买入,但趋势已坏、继续跌。 事件闸门专门识别这种坑——关键是「因果」不是「时间近似」。

触发买入的这次下跌来源20日涨率保守下界平均收益
⚠️ 财报暴雷砸的坑(财报日大阴线<−6%)62%52%+3.3%
财报附近但温和回落(−6%~0)71%59%+7.1%
远离财报(>30 天)71%68%+5.6%
买入信号总样本 N=1213、基准涨 70%。注意第 2、3 行几乎一样:离财报近但只是温和回落(71%),和远离财报(71%)一样好——有毒的只有财报当天/次日大阴线砸出的坑(62%,下界仅 52%)。所以闸门只在真雷区亮红旗,温和回落/远离财报不误伤。
⚠️ 财报暴雷坑近 15 天有财报日 且 财报当日/次日单日 < −6%
买入信号照常触发,但标注 ⚠️「财报暴雷坑(距财报X天·当日−X%·历史62%,慎接飞刀)」——由你自行判断这是错杀还是真雷。不硬拦,只提示。

🧪 为什么只留事件闸门

Regime / 质量 / 事件 三个专业量化闸门,先回测再决定——数据否掉了两个
✗ Regime 过滤大盘 SPY vs 200 线:反向——熊市抄底 73% 反而高于牛市抄底 65%。超卖信号在熊市更接近真底、均值回归更强,加"只在牛市买"会滤掉更好的机会。不加。
✗ 质量闸门ROE / 杠杆:无区分度——ROE 健康(>12%)涨 72% vs ROE 低涨 71%,几乎无差。池子是 31 只精选优质大盘股,天然没有价值陷阱可过滤。不加。
✓ 事件闸门财报暴雷坑:唯一有效——真雷区 62% vs 常态 71%,差 9-12 个百分点。已落地。
认知:这些通用闸门是为"全市场、含垃圾股、跨牛熊"设计的。而我们的池子是精选优质大盘股——已经天然过滤了低质量标的,且超卖信号跨 regime 都有效,所以通用闸门在这个特定池子上失效。先测再落地,省掉两层没用的复杂度。

为什么 KDJ / MACD 没进大机会信号

回测过 KDJ 超卖/金叉、MACD 金叉/零轴下金叉/底背离——两组的独立 edge 都在 ±2% 内,没有可交易价值。它们只保留在工具的「看法」列做粗略投票(那本就不追求 edge)。量价配合、背离作为独立方向信号 edge 也弱,同样未纳入。

数据来源　gen_signal_config.py 对 31 标的近 5 年日线 per-ticker 校准 RSI 阈值 + 布林周期,判定用原始胜率(多持有期达标)而非 Wilson 下界(涨=触发后 N 日上涨概率)。RSI 为日线 14 周期(Wilder 平滑)。
个股回溯率(悬浮/邮件显示)　每条信号附该标的自己的历史回溯,给 5日 / 10日 / 20日 三个持有期的方向胜率(同一批信号、以20日valid为准)——看反弹节奏而非只看终点。如 GEV 破50日布林:先反弹再探底再回正;MCD 强买入 100/100/100。N 小甚至=1 也保留:由 N 自证其罕见=极端,不冒充 edge。
事件闸门数据源　财报公布日取 /vX/reference/financials 的 filing_date——季报(10-Q)精确,年报季(10-K)常缺 → 用 end_date+50 天近似(标"近似");单日跌幅用日线现算、100% 可靠。云端限速拉不到财报时标"财报日期未知"、不拦买入。
已知盲区(财报当天)　财报当天/次日,新季 filing_date 还没进数据源(T+1~数天才入库),此刻最该警示的"财报当天暴雷"反而认不出——而 Starter 档拿不到前瞻财报日历(Benzinga 端点 403)。**不用"当天大阴线"兜底**:回测显示纯大阴线抄底反而更准(单日 −8% 达 80%),只有"财报造成的"大阴线才有毒,兜底会误伤好机会。折中:当距最近已知财报 >85 天(下季可能已发未入库)且触发买入时,附**中性提示 ℹ️「财报数据仅到 X,请自行核对是否刚发财报」**——不标坑、不改判断、不误伤,只提醒你多看一眼。
落地位置　① 选股工具「机会」列(悬停看依据,踩坑标 ⚠️);② 每日邮件告警(每交易日盘中 3 次扫描,有触发才推,买入信号带 ⚠️ 财报暴雷坑标注)。工具 / 告警 / 回测三处口径完全一致。
方法论　per-ticker 阈值/周期 + 事件闸门都靠数据回溯、非拍脑袋。⚠️ 阈值在同份 5 年数据上选 + 评估,有过拟合乐观偏差——小样本高胜率作"极端信号提示"、别当可交易 edge,实盘打折看待。
